What is a variable?

A variable is a letter that stands for a number we don’t know yet. It’s not a mystery, it’s just a placeholder.

Examples

If x + 3 = 7, then x must be 4.

If 2a = 10, then a must be 5.

Common mistakes

  • Thinking variables change randomly
  • Treating letters like units instead of numbers

Practice

  1. x + 5 = 9
  2. 3a = 12
  3. y − 2 = 6

Answers

1) x = 4    2) a = 4    3) y = 8
